Receita da Boa - Plataforma Social de Culinária
O Receita da Boa é uma aplicação web full-stack desenvolvida para facilitar o compartilhamento de conhecimentos culinários, permitindo que usuários publiquem suas receitas, interajam com outros cozinheiros e participem de uma comunidade moderada e organizada.

Este projeto foi desenvolvido como parte da avaliação do Curso de tecnologia da fatec praia grande, focando em integração full-stack e modelagem de dados relacional.

--------------------------------
Entusiastas de culinária muitas vezes sentem falta de uma plataforma centralizada que una a organização pessoal de receitas com o aspecto social de seguir autores favoritos e descobrir pratos por categorias, tudo isso sob um sistema de moderação que garanta a qualidade do conteúdo.

----------------------------------

Tecnologias Utilizadas (Stack)
Frontend: React.js (Interface reativa e componentizada)

Backend: Node.js com Express (API RESTful)

Banco de Dados: PostgreSQL (Persistência de dados relacional)

Ambiente de Desenvolvimento: Replit (CDE para desenvolvimento e deploy contínuo)

Funcionalidades Principais
 Autenticação Segura: Integração com Google OAuth.

 Gestão de Receitas: CRUD completo de receitas com categorias.

 Rede Social: Sistema de seguidores, curtidas e comentários em tempo real.

 Painel Administrativo: Ferramentas de moderação para análise de denúncias.
